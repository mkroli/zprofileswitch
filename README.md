# zprofileswitch
zsh plugin to manage multiple profiles

## Installation

### zgen
```zsh
zgen load mkroli/zprofileswitch
```

## Usage
```
zprofileswitch

Usage:
  zprofileswitch list
  zprofileswitch switch <profile>
  zprofileswitch edit <profile>
  zprofileswitch set <profile> <statement>...
  zprofileswitch delete <profile>
```
